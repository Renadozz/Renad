# RPG Adventure
A simple turn-based RPG game inspired from the game "Knights of pen & paper".

## Goals to learn
- Object-Oriented Programming (OOP)
- Classes and Objects
- Inheritance
- Lists/Arrays
- Turn-Based Combat
- Basic GUI Design

## Features
- Turn-based combat
- Leveling system
- Quests
- Shop system
- Story progression

## Programming language
- Java
- JavaFX
